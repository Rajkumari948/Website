Tags Used Inside <body> Tag

1. <h1> - Headings

Used to define headings of different levels. <h1> is the main heading. and <h3> is a sub-subheading

2. <p> - paragraph

This tag is used to write normal text or paragragh. Each <p> create a new block of text.

3.<strong> - Strong Importance

This tag highlightsb text with strong Importance. It usually appends bold.

3. <code> - Code Snippet

This tag is used to display HTMl tags or programming code in a monospace font. Useful for showing code examples.

4. <hr> - Horizontal Rule

This tag is creates a horizontal line across the page. It is used to separate sections.

5. <ul> and <li> - Unordered List and List Items

<ul> creates a bulleted list. <li> defines each item in the list
